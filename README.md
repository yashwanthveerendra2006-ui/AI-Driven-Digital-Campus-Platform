# 🎓 AI-Driven Digital Campus Platform

A responsive and user-friendly digital campus website developed using HTML, CSS, and JavaScript. The platform provides a centralized interface for students, faculty, and visitors to access academic information, course details, faculty profiles, events, and contact resources.

## 🚀 Features

- 🏠 Home Page with campus overview
- ℹ️ About Page describing the institution
- 📚 Courses Page showcasing available programs
- 👨‍🏫 Faculty Page with faculty information
- 🎉 Events Page for campus activities and announcements
- 📞 Contact Page for communication and inquiries
- 📱 Fully responsive design for mobile, tablet, and desktop devices
- ⚡ Interactive navigation and dynamic UI elements using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Media Queries
